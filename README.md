# Тестовое задание

## Как запустить

1. Запустите сервер:

    ```bash
    ./serveRun.sh
    ```

2. Подождите, пока Maven запустит сервер.
3. После успешного запуска сервера, запустите клиент:

    ```bash
    ./clientRun.sh
    ```

> **Важно**: если запустить клиент, пока сервер ещё не запустился — крашнет.
